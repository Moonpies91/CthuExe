4.3 Bonding Curve Mathematics
Model: Exponential bonding curve (Pump.fun style)
// Price calculation
price = virtualReserve / (totalSupply - tokensSold)

// Buy calculation
tokensOut = tokensSold * (1 - (virtualReserve / (virtualReserve + monadIn)))

// Sell calculation
monadOut = virtualReserve * (1 - ((totalSupply - tokensSold) / (totalSupply - tokensSold + tokensIn)))
Virtual Reserve Initial Value: Calibrate to achieve ~$0.00001 starting price per token

12. SECURITY REQUIREMENTS
12.1 Smart Contract Security
	•	ReentrancyGuard on all external functions with transfers
	•	SafeERC20 for all token operations
	•	Solidity 0.8.24+ for built-in overflow protection
	•	No delegatecall to external contracts
	•	No proxy patterns - all contracts immutable
	•	No owner functions, no pause functions
	•	No migrator pattern in farm
